# Tests

### Setup

- Setup docker before running integration tests.

### Unit tests

Run

```sh
pnpm test
```

### Integration tests

- run integration tests

```sh
pnpm test:integration
```

### Integration tests

```sh
pnpm test:e2e
```

### Watch mode

Recommended using watch mode to simplify writing new tests.

Run

```sh
pnpm test:watch
```
